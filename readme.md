## Geografun
Merupakan Sebuah Aplikasi Pembelajaran yang Berbasis Web untuk belajar tentang Geografi Dunia melalui serangkaian Fun Quiz


**Software Yang Digunakan Pada Pekerjaan Ini :**

```
- MySQL + GORM
- Echo Go
- Docker
- Postman
- Swagger
- Visual Studio Code
```

**Lainnya :**

